Biography & profile photo

To edit the biography text: Open index.html in a text editor. Find the section marked <!-- EDIT YOUR BIOGRAPHY TEXT BELOW --> and change the text between the <p> tags. Save the file and upload to GitHub.
To change the profile photo: Prepare a portrait image (JPG, ideally 600×800px or larger). Name it profile.jpg and upload it to the images/profile/ folder on GitHub, replacing the existing file.
To change the hero background image (the large image on the home page): Name your image hero-bg.jpg and upload it to images/profile/, replacing the existing file.
To edit details (location, medium, commission status): In index.html, find the bio-details section and update the text inside the <p class="bio-detail-value"> tags.

Reviews & quotes

To update review platform links: Open reviews.html. Find each <a href="..."> in the platform cards and replace the href with your actual Google, Facebook, or Etsy review page URL.
To add or change featured quotes: In reviews.html, find the <!-- EDIT: Replace these with real quotes --> comment and edit the quote cards below it.

Contact details & address

Contact details (email, address) appear in the footer of every page and on contact.html and privacy.html. Search each file for jane@example.com and TR18 XXX and replace with real details. Also update the mailto: links in buy.html and contact.html.

Setting up the contact form (one-time)

Action needed

Currently the contact and buy forms use a mailto: link as a temporary fallback — this opens the visitor's email app, which isn't ideal. To get proper form submissions delivered to your inbox, sign up for Formspree (free for up to 50 submissions/month):

  1. Go to formspree.io and create a free account with your email address.
  2. Create a new form — Formspree will give you a unique URL like https://formspree.io/f/xyzabc12
  3. In buy.html and contact.html, find the commented-out OPTION A: Formspree section and uncomment it, replacing YOUR_ID with your Formspree form ID.
  4. Remove or comment out the mailto: fallback lines below it.
  5. Upload the changed files to GitHub and test by submitting a test enquiry.

How to publish changes to GitHub

All changes to the site are made by uploading or editing files on GitHub. Here's the simplest method using the GitHub website (no technical tools needed):

  1. Go to github.com and log into your account.
  2. Click on your repository (it will be named something like yourdomain.github.io).
  3. To upload a new image: Navigate to the correct folder (e.g. images/gallery/), click Add file → Upload files, drag your image in, scroll down and click Commit changes.
  4. To edit a JSON file (adding/removing works): Click on the file (e.g. data/forsale.json), then click the pencil icon ✏️ to edit it. Make your changes, then click Commit changes.
  5. To edit an HTML file (biography text, links, etc.): Click on the file, then click the pencil icon ✏️ to edit. Find the text to change, update it, click Commit changes.
  6. Wait 1–2 minutes. GitHub Pages rebuilds automatically. Refresh your website to see the change.
